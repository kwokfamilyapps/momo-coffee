📓 工具 1：Colab Notebook（抽 GPS → CSV）

Google Colab 係免費嘅網上 Python 執行環境，喺 browser 行，唔使安裝任何嘢。
下面係完整 Notebook 內容，直接 Copy → Paste 去 Colab。

點樣用？

  1. 去 https://colab.research.google.com
  2. 㩒 File → New Notebook
  3. Copy 下面嘅 code → Paste 入 Colab 嘅 cell
  4. 㩒 Play ▶️ 逐格 Run

完整 Code（Copy 晒佢）：

# @title 1. 安裝 HEIC 支援（㩒 Play Run） !pip install pillow-heif -q print("✅ HEIC 支援已安裝") # @title 2. Mount Google Drive（授權連接） from google.colab import drive drive.mount('/content/drive') print("✅ Google Drive 已連接") # @title 3. 🔧 改呢行 — 你嘅相片 Folder 路徑 FOLDER_PATH = "/content/drive/My Drive/山徑相片/2026-07-23" # ↑ 改做你放相嘅 Google Drive 路徑 # @title 4. ⏳ Run 抽 EXIF + 出 CSV import os, csv from PIL import Image from pillow_heif import register_heif_opener register_heif_opener() def dms_to_dd(dms, ref): if not dms or len(dms) < 3: return None dd = float(dms[0]) + float(dms[1])/60 + float(dms[2])/3600 return round(-dd if ref in ('S','W') else dd, 6) results = [] for root, dirs, files in os.walk(FOLDER_PATH): for f in sorted(files): if not f.lower().endswith(('.heic','.jpg','.jpeg','.png')): continue fp = os.path.join(root, f) try: img = Image.open(fp) exif = img.getexif() dt = exif.get(306, "") lat = lon = alt = None if 34853 in exif: gps = exif.get_ifd(34853) if gps.get(2) and gps.get(4): lat = dms_to_dd(gps[2], gps.get(1,b'N')) lon = dms_to_dd(gps[4], gps.get(3,b'E')) if gps.get(6): alt = round(float(gps[6]), 1) if lat and lon: results.append({'filename':f,'datetime':dt[:19], 'lat':lat,'lon':lon,'altitude':alt or ''}) print(f"✅ {f}: {lat}, {lon}") else: print(f"⚠️ {f}: 冇 GPS") except: print(f"❌ {f}: 讀唔到") print(f"\n📊 成功: {len(results)} 張有 GPS") # 分組 + 輸出 CSV from collections import defaultdict groups = defaultdict(list) for p in results: groups[(round(p['lat'],3), round(p['lon'],3))].append(p) for i, (coord, photos) in enumerate(sorted(groups.items()), 1): for p in photos: p['group'] = f"CP{i}" out_path = os.path.dirname(FOLDER_PATH) + "/gps_data.csv" with open(out_path, 'w', newline='', encoding='utf-8-sig') as f: w = csv.DictWriter(f, fieldnames=['filename','datetime','lat','lon','altitude','group']) w.writeheader() w.writerows(results) print(f"✅ CSV → {out_path}") print(f"📍 {len(groups)} 個 CheckPoint, {len(results)} 張相")

📥 Colab 完成後，去 Google Drive 下載 gps_data.csv 檔案

🤖 工具 2：Gemini Prompt（CSV → HTML 報告）

用 Gemini（https://gemini.google.com）生成 HTML 報告。

點樣用？

  1. 開 Gemini → Upload 個 CSV 檔案（㩒 + 掣）
  2. Copy 下面嘅 Prompt → Paste 落去
  3. Gemini 會輸出 HTML → Copy → Save 做 report.html

Prompt Template（Copy 晒佢）：

我 upload 咗一個 CSV 檔案，入面有山徑巡查相片嘅 GPS 數據。 每行係一張相：filename, datetime, lat, lon, altitude, group(CP編號) 請幫我做以下嘢： **1. 統計分析** - 總共有幾多張相 - 幾多個 CheckPoint（CP） - 海拔範圍 - 每個 CP 有幾多張相 **2. 生成一份完整 HTML 報告** 用香港繁體中文，手機 friendly（responsive），風格簡潔。 HTML 要包含： a) 統計摘要卡片（總相數、CP數、海拔範圍、日期） b) Leaflet.js 互動地圖（用 OpenStreetMap） - 每個 CP 用 circleMarker，半徑 = 相片數量 × 1.5 - 顏色根據相片數量：少=綠色，中=橙色，多=紅色 - 㩒落去彈出 CP 編號 + 相片數 + 海拔 - 用綠色虛線連接所有 CP 做路線 - 地圖要 fit 晒所有標記 c) 每個 CP 嘅詳細卡片 - CP 編號、坐標、相片數、海拔、時間範圍 d) Summary Table **3. 地圖嘅 Leaflet code 要完整** 用 CDN link： <link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css" /> <script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"></script> CSS 用柔和自然色調（綠色為主），背景用暖色 #f0e8d8 直接俾完整 HTML code 我，唔好 cut 任何部分。
💡 Tips：
• 如果 Gemini 輸出嘅地圖唔靚，可以叫佢「改顏色/改大啲 marker/加多啲資料」
• 如果 HTML 太長 cut 咗，叫佢「繼續，俾完整 code」
• 可以叫佢「加時間線顯示拍攝順序」

☁️ 工具 3：Google Cloud Storage（免費 Host HTML）

將 Gemini 生成嘅 HTML 放上網，義工朋友可以直接㩒 link 睇。

一次性 Setup（義工做一次就得）：

  1. 去 https://console.cloud.google.com
  2. 開一個新 Project（改個名 eg mos-trail-reports）
  3. 左邊 Menu → Cloud Storage → Bucket
  4. 㩒 Create Bucket：
    • Name: 全球唯一 eg mos-trail-2026-reports
    • Location: asia-east1（台灣，最快）
    • Storage Class: Standard
    • Access Control: Fine-grained
  5. 開 Bucket 後 → 㩒 Permissions → Grant Access：
    • New principals: allUsers
    • Role: Storage Object Viewer
    • Save → Allow public access
  6. Bucket 設定 → Static Website → 剔 Enable
    • Index page: index.html

每次出新報告：

  1. Upload Gemini 俾你嘅 HTML 上 Bucket
  2. Share 條 link：
    https://storage.googleapis.com/[bucket-name]/report.html
⚠️ 注意： Google Cloud 第一次用要綁信用卡（但唔會收錢，5GB 永遠免費）。
如果你哋唔想綁卡，可以用 Firebase Hosting（都係免費）或者 GitHub Pages（免費，要 GitHub account）。
最簡單替代：直接將 HTML 放上 Google Drive → 用 htmlpreview.github.io 睇都得。

⚠️ 但要誠實講：呢個方案嘅真正限制

1. Gemini 生成 Leaflet 地圖嘅質素
Gemini 比 Claude 弱啲 — 生成嘅 Leaflet 地圖有時會 missing markers、顏色錯、或者 code 太長被 cut。可能要 prompt 幾次先改好。
建議： 叫 Gemini 先出「簡化版地圖」，確認 marker 位置正確後先叫佢加 details。
2. Colab 嘅 HEIC 支援
pillow-heif 喺 Colab 可以安裝，但有時版本衝突。如果 run 唔到，可以叫 Gemini 幫手 debug。
替代方案：將 HEIC 轉做 JPEG（iPhone 設定 > 相機 > 格式 > 最相容）先 upload。
3. Google Cloud Storage Setup
義工朋友第一次 setup Cloud Storage 要綁信用卡（免費 quota 唔會扣錢）。有人可能會抗拒。
替代方案： Firebase Hosting（免費唔使卡）/ GitHub Pages（免費） / htmlpreview（直接 Google Drive link 就得）
4. 唔係完全自動化
義工仍然要手動：copy Colab code → run → download CSV → upload Gemini → paste prompt → copy HTML → upload Cloud。
大約 10-15 分鐘嘅手動操作。
