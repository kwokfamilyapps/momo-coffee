1 📸 準備數據

每個 CheckPoint 嘅相整理成 CSV，格式係咁：
filename,datetime,lat,lon,altitude,group IMG_1765.HEIC,2026-07-23 10:27:00,22.400197,114.246644,309,CP1 IMG_1768.HEIC,2026-07-23 10:52:51,22.398025,114.248436,359,CP1 IMG_1774.HEIC,2026-07-23 12:20:15,22.395411,114.249533,423,CP2
點樣得到呢個 CSV？有兩個方法：
方法 A：用 Colab Notebook（自動抽 GPS，見下面）
方法 B：自己用 Excel 打（如果知坐標）
📓 Colab Notebook（Copy & Paste 去 colab.research.google.com）：
# @title 1. 安裝 HEIC 支援 !pip install pillow-heif -q # @title 2. Mount Google Drive from google.colab import drive drive.mount('/content/drive') # @title 3. ✏️ 改呢行 — 你放相嘅路徑 FOLDER_PATH = "/content/drive/My Drive/山徑相片/2026-07-23" # @title 4. Run 抽 GPS → CSV import os, csv; from PIL import Image; from pillow_heif import register_heif_opener; register_heif_opener() from collections import defaultdict def dms_to_dd(d, r): return round(float(d[0])+float(d[1])/60+float(d[2])/3600, 6) if d and len(d)>2 else None results = [] for r, d, fs in os.walk(FOLDER_PATH): for f in sorted(fs): if not f.lower().endswith(('.heic','.jpg','.jpeg')): continue try: img = Image.open(os.path.join(r,f)); ex = img.getexif(); dt = ex.get(306,"") lat=lon=alt=None if 34853 in ex: g = ex.get_ifd(34853) if g.get(2) and g.get(4): lat=dms_to_dd(g[2],g.get(1,b'N')); lon=dms_to_dd(g[4],g.get(3,b'E')) if g.get(6): alt=round(float(g[6]),1) if lat and lon: results.append({'filename':f,'datetime':dt[:19],'lat':lat,'lon':lon,'altitude':alt or '','group':''}) except: pass groups = defaultdict(list) for p in results: groups[(round(p['lat'],3),round(p['lon'],3))].append(p) for i,(c,ps) in enumerate(sorted(groups.items()),1): for p in ps: p['group']=f'CP{i}' out = os.path.dirname(FOLDER_PATH)+"/gps_data.csv" with open(out,'w',newline='',encoding='utf-8-sig') as f: w=csv.DictWriter(f,fieldnames=['filename','datetime','lat','lon','altitude','group']); w.writeheader(); w.writerows(results) print(f"✅ Done! {len(results)} photos, {len(groups)} CPs → {out}")

2 🤖 將 CSV + Prompt 俾 Gemini

📋 Prompt Template（Copy 晒，改 {黃色字}）

我 upload 咗一個 CSV 檔案，係 {馬鞍山郊遊徑} 嘅巡查相片 GPS 數據。 📊 基本資料： - 日期：{2026年7月23日} - 總相片：{XX} 張 - CheckPoints：{X} 個 - 海拔範圍：{XXX-XXX} 米 - 拍攝工具：{iPhone 12 Pro Max} 🗺️ 請幫我生成一份完整 HTML 報告，手機 friendly： 1. 統計摘要卡片（總相數、CP數、海拔、日期） 2. Leaflet.js 互動地圖，用 OpenStreetMap tiles： - 每個 CP 一個 circleMarker，半徑 = 相片數 × {1.5} - 顏色：少相=🟢綠色，中相=🟠橙色，多相=🔴紅色 - 㩒 marker 彈出 CP 編號 + 相片數 + 海拔 - 用 {綠色虛線} 連接所有 CP 做路線 - 地圖 fit 晒所有標記 3. 每個 CP 詳細卡：編號、坐標、相片數、海拔、時間範圍 4. Summary Table 額外要求： - {（可以加其他要求，例如：顏色主題、加時間線等）} 風格：溫暖自然色系，背景 #f0e8d8，綠色為主。 直接俾完整 HTML code。
操作步驟：
1️⃣ 開 https://gemini.google.com
2️⃣ 右上角 Upload CSV 文件（㩒 + → Upload）
3️⃣ Copy 上面 Prompt → Paste 落去
4️⃣ 改晒 {黃色字} 嘅內容
5️⃣ Enter → Gemini 會輸出 HTML
6️⃣ Copy HTML → 貼上記事本 → Save 做 {report-20260723}.html

3 ☁️ 放上網俾人睇

最簡單方法（唔使 setup，即時有）：

Option A：htmlpreview（最快）

https://htmlpreview.github.io/? + 你條 Google Drive link

1. 去 console.cloud.google.com → Cloud Storage
2. Upload HTML → Permissions → allUsers → Storage Object Viewer
3. 得到 link：
https://storage.googleapis.com/{bucket-name}/{report-20260723}.html

💡 進階 Tips：
• 如果同一條路線行幾次，可以叫 Gemini 將新舊數據 merge 做「對比報告」
• 如果想加「清理前後對比」描述，喺 CSV 加多個 column "status"（before/after）
• 如果 Gemini 出嘅 map 唔靚，叫佢「用多啲顏色/加大啲 marker/加多啲資料」
• 可以叫 Gemini 加「路線海拔圖」（用 Chart.js）— 但呢個要睇佢生唔生成到
