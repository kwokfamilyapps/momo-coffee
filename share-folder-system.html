📂 檔案結構（義工只需咁樣放）

Google Drive 嘅 Shared Folder 📁 └── 📁 馬鞍山郊遊徑 ← Folder 名 = 路線名 ├── 📁 清理水道前後 ← Subfolder 分類（可選） │ ├── IMG_1768.HEIC │ ├── IMG_1774.HEIC │ └── ... └── 📁 山徑巡查 ├── IMG_1785.HEIC └── ...
✅ 義工只需要：
1. 行山影相（GPS auto embedded）
2. 將相放入 Google Drive folder
3. Folder 名改做路線名 eg「馬鞍山郊遊徑」
4. Right-click → Share → Anyone with link → Copy link
5. 將條 link 俾義工隊長（或者直接俾 Gemini）

🛠️ 方案 A：隊長用 Colab 做（最簡單自動化）

隊長收到義工嘅 Share link 後，用 Colab Notebook 一鍵處理：

📓 一鍵 Colab Notebook（製作中）
One-click Colab 會自動做晒：
✅ 貼 Share link → 自動解析 Folder ID
✅ Mount Google Drive → 讀相片
✅ 認 Folder 名做路線名（「馬鞍山郊遊徑」）
✅ 認 Subfolder 名做分類（「清理水道前後」「山徑巡查」）
✅ 抽 EXIF GPS + 日期 + 海拔
✅ 按坐標分 CheckPoint
✅ 輸出 CSV（內置路線名 + 分類資料）
✅ CSV 自動 save 返落 Google Drive

🤝 方案 B：義工 Share 俾 Ming → Mvmv 幫手

📋 Gemini Prompt（義工隊長版）

收到 CSV 後，Copy 呢段 Prompt + Upload CSV 俾 Gemini：

我 upload 咗一個 CSV 檔案，係 {路線名} 嘅巡查 GPS 數據。 請幫我生成一份手機-friendly HTML 報告，要包括： 1️⃣ 統計摘要：總相數、CheckPoint 數、海拔範圍 2️⃣ Leaflet 地圖：每個 CP 用圓形 marker，大細 = 相片數 - 用綠色虛線連接所有 CP - 㩒 marker 彈出 CP 資料 3️⃣ 每個 CP 詳細列表 4️⃣ Summary Table 風格：暖色背景 #f0e8d8，綠色為主 直接俾完整 HTML code
